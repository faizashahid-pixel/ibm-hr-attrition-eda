# IBM HR Attrition EDA

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the IBM HR Employee Attrition & Performance dataset to identify key factors influencing employee turnover.

##  Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

##  Key Analysis
- Attrition distribution
- Income vs Attrition
- Age, Job Role, and Work-Life Balance analysis
- Gender, Distance, Marital Status Etc

## Dataset
IBM HR Employee Attrition & Performance (Kaggle)

## Conclusion
The analysis highlights important factors contributing to employee attrition, which can help organizations improve retention strategies.
